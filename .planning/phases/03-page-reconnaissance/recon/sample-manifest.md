## Notes

1. **URL-to-match-ID mapping:** The match IDs in the HLTV URL path do not always match what the user intended when constructing URLs manually. The actual match content is determined by the numeric match ID, not the slug. For example, `/matches/2367432/vitality-vs-faze-...` actually loaded 9 Pandas vs FORZE because 2367432 is the real match ID for that game.

2. **Performance page size variance:** performance-206389.html.gz (218,969 chars) is notably smaller than other performance pages (~6.2M chars). It still contains valid stats tables and overview data. The size difference may be due to reduced navigation chrome or ad content -- the page is still fully functional for analysis.

3. **Forfeit match structure:** match-2380434 has map name "Default" and zero mapstatsid links, confirming that forfeit matches have no map statistics pages at all. The overview page itself is slightly shorter than average but still substantial (6.1M chars).

4. **BO5 with partial forfeit:** match-2384993 is an unusually rich sample. It's a BO5 grand final where the 3rd map was forfeited (shows "Default"), the 1st map went to overtime (19-17), and only 2 of 5 maps have stats links. This captures multiple edge cases in a single match.

5. **All pages appear to use Rating 3.0:** The `standard-headline` elements on map stats pages contain "Performance - rating 3.0" text across all eras, confirming the retroactive Rating 3.0 application documented in 03-RESEARCH.md.
